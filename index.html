・the button element

(1)説明

ボタンを作るための要素である。 ボタンをクリックすることで、任意の動作をつけることができる。 ボタンの外見を変えたい時は、CSSを使うと変えることができる。

(2)いくつかのアークアップ例


            <!DOCTYPE html>
            <html>
                <head>
                    <meta charset="utf-8">
                    <title>テスト１</title>
                </head>
                <body>
                    <button>button</button>
                </body>
            </html>

・the title element

(1)説明

Webページのタイトルを表す要素である。 ページのタブや、検索されたときに出るサイト名を表す要素である。 テキスト以外は記入できないことに注意しなければならない。 Webページを開いたときに、１番上にある見出しのようなものとは別なので注意しなければならない。

(2)いくつかのアークアップ例


            <!DOCTYPE html>
            <html>
                <head>
                    <meta charset="utf-8">
                    <title>ここがtitle要素で変わる部分である</title>
                    </head>
                <body>
                    
                </body>
            </html>

・the h1, h2, h3, h4, h5, and h6 element

(1)説明

<h1>,<h2>,<h3>,<h4>,<h5>,<h6>は見出しを表す要素である。 <h1>が最も主要な見出しである。数字が小さいほど主要な見出しとなる。 主要な見出しであるほど字のサイズは大きくなる。 基本的に番号の小さいタグから使うべきである。

(2)いくつかのアークアップ例

            <!DOCTYPE html>
            <html>
                <head>
                    <meta charset="utf-8">
                    <title>テスト3</title>
                </head>
                <body>
                    <h1>これが<h1>タグで書いた見出しである。</h1>
                    <h2>これが<h2>タグで書いた見出しである。</h2>
                    <h3>これが<h3>タグで書いた見出しである。</h3>
                    <h4>これが<h4>タグで書いた見出しである。</h4>
                    <h5>これが<h5>タグで書いた見出しである。</h5>
                    <h6>これが<h6>タグで書いた見出しである。</h6>
                </body>
            </html>>
